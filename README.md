## 실행 방법
1. 클라이언트
- IDE 터미널에서 node -v 하여 버전 정보 확인 후 없으면 node.js 설치
- emr-react로 이동하여 npm install
- npm start 입력하여 시작

2. 서버
- application.properties 파일에서 본인의 로컬 MariaDB 아이디, 비밀번호 입력 후 세팅

## 주요 기능  
- JWT 인증 및 보안  
- 리액트를 활용한 프론트엔드 개발

## 구현 목표
- 병원 응급실 정보를 실시간으로 제공
